## Examples

<Callout type="warning">
  **`Radio`** is a primitive component, it is not meant to be used on it's own.
  It should always be be wrapped in a **`RadioGroup`** and accompanied by a
  **`Label`**.
</Callout>

### Default

The default **`Radio`** uses size `md`.

<ComponentExample name="radio" story="default" />

### Sizes

You can use the `size` prop to alter the **`Radio`** size. Sizes `sm`, `md`, and `lg` are included by default.
