## Examples

<Callout type="important">
  **`Checkbox`** is a primitive component, it is not meant to be used on it's
  own. It should always be be wrapped in a **`CheckboxGroup`** and accompanied
  by a **`Label`**.
</Callout>

### Default

The default **`Checkbox`** uses size `md`.

<ComponentExample name="checkbox" story="default" />

### Sizes

You can use the `size` prop to alter the **`Checkbox`** size. Sizes `sm`, `md`, and `lg` are included by default.
